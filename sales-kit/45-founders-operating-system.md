# Founder&apos;s Operating System

The daily / weekly / monthly / quarterly / annual cadence that
integrates all 44 sales kit docs into one operating rhythm.

**Goal**: a printable, follow-able rhythm that prevents skipped
work + makes the sales kit actually USED.

---

## Daily rhythm (5 min + 30 min = 35 min total)

### Morning (5 min — first thing after coffee)

- [ ] Check critical alerts (Stripe disputes, security alerts,
      customer-facing site uptime)
- [ ] Glance at inbound (any Tier A leads in last 12 hours?)
- [ ] Top 1 priority for today (write it on a post-it)

### Block 1: 30-min outbound (Mon-Fri rotating)

| Day | Activity | Source doc |
|---|---|---|
| Mon | LinkedIn outbound (10 connections + 5 DMs + 3 bumps) | 21 |
| Tue | Blog post + 1 short-form video | 18, 26 |
| Wed | LinkedIn post + reply to comments | 27 |
| Thu | Reddit / Quora answer (1 post) | 18 |
| Fri | Newsletter (write + schedule) | 27 |

Pick today&apos;s block based on day-of-week. Don&apos;t skip.

### Triage block (continuously, but cap at 60 min/day)

- Reply to inbound leads per Tier rules (Tier A: 1h, Tier B: 4h)
- Use templates from doc 31

### End of day (5 min)

- [ ] Log today&apos;s actions in tracking sheet
- [ ] Note: 1 thing that worked today, 1 thing that didn&apos;t
- [ ] Pre-pick tomorrow&apos;s priority

**Total daily time: ~35-90 minutes.** Don&apos;t skip days. If
running short, skip the optional triage block (other than urgent
Tier A) — never skip the Mon-Fri rotating block.

---

## Weekly rhythm (30 min, Friday end-of-day)

### Friday 4-5pm: Weekly review (sales kit doc 32)

- [ ] Pull 12 metrics from tracking spreadsheet
- [ ] Compare to last week (improving / flat / declining)
- [ ] Compare to 90-day target (on track / behind)
- [ ] Note: 1 thing to change next week
- [ ] Save row in spreadsheet

### Friday 5-5:30pm: Customer health check

For top 5 customers by revenue:
- [ ] Recent activity (login, support tickets, mail volume)
- [ ] Health score (per doc 39 framework)
- [ ] Anyone need outreach?

If any customer scores poorly: schedule a Monday-morning call.

### Weekend: REST

This is non-negotiable. Burnout is the #1 reason solo execution
fails.

---

## Monthly rhythm (2 hours, last Friday of month)

### Block 1: Financial review (45 min)

- [ ] Pull P&L from QuickBooks
- [ ] Reconcile accounts (per doc 43)
- [ ] Variance analysis (actual vs forecast)
- [ ] Cash-flow check (next 30-60 days outlook)

### Block 2: Cohort + retention review (30 min)

- [ ] Pull cohort report (% of [signup month] still active by
      product line)
- [ ] Calculate churn rate this month
- [ ] Compare to last 3 months

### Block 3: Best-piece-of-content review (15 min)

- [ ] Highest-performing piece of content this month
- [ ] Lowest-performing piece
- [ ] Replicate the winner format (+ make a template)
- [ ] Stop doing the loser format

### Block 4: Plan next month (30 min)

- [ ] Set 3 priorities for next month
- [ ] Block calendar for big items
- [ ] Adjust content calendar (doc 30) if needed

---

## Quarterly rhythm (4 hours, last Friday of each quarter)

### Block 1: Quarterly metrics review (60 min)

- [ ] Pull 13-week trend on each of the 12 metrics
- [ ] Best week vs worst week — what was different?
- [ ] What worked / what didn&apos;t?
- [ ] What&apos;s 1 thing to change next quarter?

### Block 2: Strategic priorities check-in (60 min)

- [ ] Review the 3-5 priorities from the annual plan (doc 34)
- [ ] Hit / miss / adjust each
- [ ] Add 1 emerging priority if relevant
- [ ] Drop 1 priority that&apos;s become obsolete

### Block 3: Customer interviews (60 min)

- [ ] Schedule 3-5 customer calls (mix of happy / churned / new)
- [ ] 30-min calls — listen more than talk
- [ ] Compile patterns into a 1-page memo

### Block 4: 1-page quarterly memo (60 min)

Save in `/sales-kit/_quarterly-reviews/[YEAR]-Q[N].md`. Use
template from doc 32.

---

## Annual rhythm (4-day session, late December)

### Day 1: Reality check (4 hours, doc 34 Section 1-2)

- [ ] Pull last year&apos;s actual revenue + costs
- [ ] Honest filter (cash runway, customer concentration, margin
      trends)
- [ ] Five-engine review (score 1-10 on growth / profitability /
      strategic value)

### Day 2: Strategy (4 hours, doc 34 Section 3-4)

- [ ] 1-year vision
- [ ] 3-5 strategic priorities + quarterly milestones
- [ ] 3-5 things we WON&apos;T do this year

### Day 3: People + budget (4 hours, doc 34 Section 5-7)

- [ ] Revenue scenarios (conservative / base / stretch)
- [ ] Budget for chosen scenario
- [ ] People plan (hires this year, per doc 36)
- [ ] Top 3 risks + contingencies

### Day 4: Team kickoff (4 hours, only if you have a team)

- [ ] Share the 3-page memo
- [ ] Assign owners + cadence
- [ ] Define success per owner

Solo? Skip Day 4 — the plan is yours; review weekly through the
year.

## When unexpected events happen

### Crisis (per doc 42)

Drop daily / weekly rhythm. Activate crisis playbook for the type
of event. Resume normal rhythm only after crisis is resolved or
contained.

### Big opportunity (e.g., partnership inquiry, acquisition outreach)

- [ ] Take 24 hours to think before responding
- [ ] Run the math (does this fit our priorities?)
- [ ] Decide: pursue / decline / explore lightly
- [ ] If pursuing: temporary 50/50 split between BAU and the
      opportunity until it lands or doesn&apos;t

### Burnout warning signs

- [ ] Working 60+ hours / week for 4+ weeks
- [ ] Skipping daily rhythm 3+ days / week
- [ ] Resenting customer interactions
- [ ] Losing sleep over work

Action when burnout warning: take 3-5 days off (or whatever you can
manage). Hire help (per doc 36). Cut scope of priorities.
Don&apos;t push through.

## What to skip if time is constrained

If running 50% capacity vs full schedule, skip in this priority
order:

| Priority to skip first → last |
|---|
| 1. Reddit / Quora answers (least time-sensitive) |
| 2. TikTok / Reels (one-off posts, low compounding individually) |
| 3. LinkedIn post |
| 4. Friday newsletter |
| 5. Tuesday blog post |
| 6. LinkedIn outbound |
| 7. Inbound triage |
| 8. Customer health check |
| 9. Critical alerts (NEVER skip) |

Inbound triage and critical alerts are non-negotiable. Everything
else can be paused for a week.

## Calendaring: how to make this stick

Block on your calendar (recurring):

- **30-min daily block (M-F, 9-9:30am)**: outbound work
- **30-min Friday 4-4:30pm**: weekly review
- **2 hours last Friday of month, 1-3pm**: monthly review
- **4 hours last Friday of quarter, 9am-1pm**: quarterly review
- **4 days late December (e.g., Dec 26-29)**: annual planning

Treat these as appointments. If a customer asks for a meeting in
your block, push them to a different time. The cadence matters
more than any single meeting.

## What this OS prevents

Without this OS, founders typically:

1. Skip outbound for "more important" work — top of funnel dies
2. Skip weekly metrics — only realize problems quarterly
3. Don&apos;t do customer interviews — make decisions on assumptions
4. Don&apos;t plan annually — operate reactively
5. Don&apos;t have a crisis playbook — panic when something breaks
6. Don&apos;t have books in order — discover financial issues at
   tax time
7. Don&apos;t have data room ready — can&apos;t exit cleanly
8. Burn out — quit altogether

This OS prevents all of these. It&apos;s overhead in the short
term and leverage in the long term.

## Tracking your adherence

Weekly self-score (1-10):

- Did I do daily 30-min block 5 days this week? __ / 10
- Did I respond to Tier A leads within 1 hour? __ / 10
- Did I do my Friday review? __ / 10
- Am I on track for monthly priorities? __ / 10

Average score < 6 for 2+ weeks: the OS isn&apos;t working as
designed. Either:
- The schedule is too aggressive (cut scope)
- You&apos;re consistently being interrupted (defend calendar
  better)
- Something else is broken (diagnose)

Fix before pushing harder.

---

## What this OS ISN&apos;T

- **Not a substitute for product / customer focus.** The OS is the
  CONTAINER for your work, not the work itself.
- **Not bureaucracy for bureaucracy&apos;s sake.** Every rhythm has
  a specific output. If a rhythm produces nothing useful, drop it.
- **Not eternal.** Revisit the OS quarterly. Adjust as the business
  scales.

---

## The big idea

You can have the best playbooks in the world (we have 44 of them).
None of them work if you don&apos;t have a cadence to execute them.

The OS is the cadence.

The cadence is what separates the founders who hit $5M MRR from
the ones who burn out at $200k MRR.

It&apos;s not motivation. It&apos;s system. The system protects
you from yourself.

Pick a Sunday. Calendar it. Run the OS for 90 days.

Then look back. The compounding is real.
