## Production notes

**Subtitle requirement**: 100% of short-form is captioned. Use TikTok&apos;s
auto-captions (good enough — review for typos). For IG, use CapCut or
Descript before upload.

**Music**: Trending sounds drive 2-5x more reach on TikTok. Pick a
trending sound and adjust pacing to match.

**Posting times**: TikTok 7pm-10pm PT, IG 11am-2pm + 7pm-9pm PT. Consistent
times beat random.

**Repurposing**: Each TikTok can be:
- Re-uploaded to IG Reels (often with the same caption)
- Re-uploaded to YouTube Shorts
- Embedded in blog posts (relevant content matches)
- Sent in welcome emails (auto-play disabled per email best practices)

**Affiliate adaptation**: Affiliates rewrite the CTA: "use my code [CODE]
for the same deal — link in my bio."

**Compliance**:
- No medical, legal, tax claims as guaranteed advice (always say "talk to
  a CPA / attorney")
- No before/after revenue claims that aren&apos;t real
- No claims that we save money in cases where state laws make it
  uncertain

**A/B testing**:
- Run 5 hooks on the same body content. Pick the winning hook.
- Test 15s vs 30s on the same content. Shorter wins for awareness, longer
  wins for conversion.
- Test "you" vs "they" framing. "You" wins for sales-focused content.
